# Simple Map Template

HTML, CSS, JavaScript のみで作るシンプルな地図アプリのテンプレート

## 構成

- HTML
- CSS
  - Tailwind の導入を検討
- JavaScript
  - OpenStreetMap (Leaflet.js)

## 動作確認

- index.html を開く
  - Leaflet がエラーを出したら `python -m http.server 8000` でローカルサーバで起動
